IoT-Based Smart Farm Monitoring System

This project presents the design and implementation of an Internet of Things (IoT)–based Smart Farming System aimed at improving agricultural productivity through real-time monitoring of soil and environmental conditions. Sensor data--including soil moisture, soil pH, temperature, humidity, fertilizer type, and crop yield--is collected, stored, and visualized using Power BI. The analysis provides insights that support precision agriculture practices, enabling optimized resource management and yield improvement.
